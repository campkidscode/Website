What We're Planning

A free, fun online camp in introductory programming skills! Students can pick from one of four paths, as shown below. In our online camp, each mentor helps teach five to six students for a month. The goal of these camps is to show students how fun and interdisciplinary computer sciencce can be! No experience in programming is necessary.

Learn programming:

  • On a flexible schedule
  • From anywhere in the world
  • With other middle-school kids!
  • Walk away with a complete website, app, etc!

Paths

CS in Art
Path 1

Learn about 3D modelling, animation, design principles, and more!

Intro to CS (general)
Path 2

For a general introduction to programming principles, we will work up to designing a game with Java.

Intro to App Design
Path 3

We will be learning how to design our own Android apps!

Intro to Web Dev
Path 4

Learn how to use HTML/CSS and Javascript to design your own interactive websites!
