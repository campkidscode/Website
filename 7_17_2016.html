Week 3 by Nithya Narayanan

  1. Home
  2. Blog Post

Posted on July 10, 2016 at 10:00 PM

In our third week at the Fanwood Library, we finished transitioning to programming in Python from the block programming we were using in trinket.io!

On Tuesday, the students learned how to use lists to store multiple values in a single variable as well as take inputs from the user and incorporate them into their programs. They used this knowledge to ask the user how many circles they wanted the program to draw and of what radius. A sample of this exercise can be found here.

On Thursday, we created a rock-paper-scissors game, which combined most of the concepts we have learned so far. A sample of this exercise can be found here. We also introduced and began work on our final projects, which will be presented next Thursday to family and friends.

The notes from Tuesday’s class can be found at the end of this powerpoint and the notes from Thursday’s class (which include links to a rock-paper-scissors sample program, the final project description, etc.) can be found here!
